# DesignSystem 活動頁延伸

> 建立日期：2026-07-13
> 類型：參考資料
> 需求 Owner：glen.wen
> 適用對象：製作無毒農檔期 / 節慶 / 新品 / 聯名 / 限時活動頁的人與 AI agent
> 資料來源：由方案 D（活動頁精煉規範）重新定位——把 D 裡「普世護欄」上收核心後，本檔只保留「活動頁專屬的少數建議 + 紅線 + 風格參考協議」
> 前置：本檔為 [核心規範](4-DesignSystem-核心.md) 的延伸。

---

## 定位（這份刻意做薄）

活動頁的重點是**搭配主題把畫面做得有趣**，而每檔活動性質不同、無法提前規範。所以：

> **活動頁的長相，由每一檔外掛的「風格參考素材」決定，不由這份文件決定。**

這份文件只做三件事：
1. **紅線**——不管素材長怎樣都不能破的線（§一）。
2. **少數風格建議**——沒有素材時的 fallback 傾向，可被素材覆蓋（§二）。
3. **風格參考協議**——要什麼素材、怎麼把素材變成該檔專屬規範（§三、§四）。

做活動頁 = **[核心](4-DesignSystem-核心.md)（全 MUST）＋ 本檔紅線 ＋ 該檔風格參考素材**。

---

## ⭐ 開工前第一件事：先拿到「風格參考素材」（權重最高，不可略過）

> **活動頁的長相不由這份 DS 決定，由你每一檔提供的風格參考素材決定。**
> DS 只給護欄與少數建議。**沒有素材，AI 只能產出偏形象頁的保守版本，做不出活動該有的氣氛。**

**開工前必備（AI 會主動跟你要，缺了不動工）：**

| 必備素材 | 說明 |
|---|---|
| moodboard / 參考風格網頁 URL | 決定整體視覺方向的主要依據 |
| 該檔主題色 | 主色 + 輔色（不取代品牌綠 CTA）|
| 關鍵視覺元素 | 主視覺圖 / 插畫風 / 質感傾向 |
| 字型傾向 | 若有；主字 Noto Sans TC 不可換 |

→ 這是活動頁流程裡**權重最高**的一步，其餘（紅線、建議）都是在素材之上的約束。完整協議見 §三。

---

## 一、紅線：就算素材那樣、客戶那樣要求，也要擋（MUST）

以下全部繼承自核心，在活動頁情境下**特別容易被「為了有趣」而想突破，這裡再明確一次**。AI agent 遇到素材或需求違反這些時，**必須擋下並向使用者說明**，不可默默照做：

| 紅線 | 對應核心 |
|---|---|
| **Logo 不可換、不可變形** | §1.6 |
| **主字型 Noto Sans TC 不可換** | §1.4 |
| **Hero 不放 CTA、標題不做按鈕樣、無連結不 hover** | §二 |
| **無障礙**：對比 ≥ 4.5:1、觸控 44px、動畫可被 reduced-motion 關閉 | §五 |
| **AI Slop 十條全禁**（漸層套版、四件組、emoji 裝飾、假數據…）——活動頁再花俏也不例外 | §七 |
| **文案品質**：不空洞金句、不行業術語、具體事實；強調大字 ≤ 40 字、不斷詞 | §四 |
| **版面紀律**：間距 4px 倍數、圓角同頁一致、同區塊對齊 | §三 |

→ 活動頁能自由的是「風格表現」，**不是**這些線。素材帶來的漸層、色彩、裝飾若踩到 AI Slop 或違反無障礙對比，一律退回。
→ **主色與 CTA 色不在紅線內**——活動頁可依主題改（見 §二），這是本次放寬。品牌識別靠 logo 與字型維持。

---

## 二、少數風格建議（SHOULD / MAY，可被素材覆蓋）

沒有素材、或素材沒指定到的地方，用這些當起點；素材有指定時以素材為準。

| 面向 | 建議 | 級別 |
|---|---|---|
| 主色 / CTA 色 | 建議優先品牌綠 `#4ba83b`，但可依主題參考改用主題色 | SHOULD |
| Section padding | 傾向比形象頁大、留白多一點，配合節奏（見 [3-版面節奏原則](3-版面節奏原則.md)）| SHOULD |
| Banner 桌機比例 | 16:9 ~ 21:9 之間 | SHOULD |
| 圓角 | 可自訂（0 也可、圓潤也可），但**同頁一致**（核心 §3.2）| MAY |
| 表單樣式 | 框線 / 底線皆可 | MAY |
| Icon 風格 | 可換（線條 / fill / 主題插畫皆可，但不踩 AI Slop 的抽象示意）| MAY |
| 懸浮元素 | 可用可不用 | MAY |
| 版型（詳情 / 圖文 / 圖片組）| 可自訂比例與交替 | MAY |
| 動態 | 可增加（scroll fade、進場動畫等），但節制、可關閉 | MAY |
| 攝影 / 主視覺 | **由風格參考素材決定**（節慶可高飽和喜氣、聯名可跟對方調性）| 依素材 |

→ 這張表故意短。活動頁不追求「一套完整預設」，追求「護欄穩、風格活」。

## 三、風格參考協議（AI agent 必做）

做活動頁時，AI **MUST** 依序執行：

1. **載入核心**（護欄，不可違背）。
2. **檢查該活動是否已有風格參考素材**：
   - moodboard / 參考風格網頁 URL
   - 該檔期主題色（幾個主色 + 輔色）
   - 關鍵視覺元素（主視覺圖、插畫風、質感傾向）
   - 特殊字型傾向（若有，仍須確認與 Noto Sans TC 的搭配，主字不可換）
3. **沒有素材 → 主動提醒使用者提供**，並說明：「活動頁的長相由素材決定，DS 只提供護欄與少數建議；不給素材我只能做出偏形象頁的保守版本。」
4. **依「護欄 + 素材」推導出該檔專屬規範**（§四模板），存到該活動資料夾 / LP brief。
5. **衝突處理**：素材 vs §二建議衝突時，**以素材為準、捨棄建議**；素材 vs 核心護欄衝突時，**以護欄為準、退回素材**並向使用者說明。

→ 此協議由 `ldp-ds` / `ldp-activity-strategy` / `ldp-asset-gen` 等 skill 觸發執行；素材規劃可搭配 [2-素材產出指引](2-素材產出指引.md)、策略搭配 [1-活動頁策略引導](1-活動頁策略引導.md)。

## 四、該檔專屬規範模板（每檔活動的產出物）

AI 依素材推導後，為該活動產出一份專屬規範，格式建議如下，存進該活動資料夾供本檔製作與同型活動複用：

```markdown
# 【活動名】專屬 DS（YYYY-MM-DD）
> 繼承：核心護欄（不可違背）+ 活動頁延伸
> 風格來源：<素材連結 / moodboard / 參考網頁>

## 主題色
- 主題主色：#______（不取代品牌綠，用於背景 / 標題 / 氛圍）
- 主題輔色：#______、#______
- CTA：維持品牌綠 #4ba83b（紅線）

## 表現值（本檔覆蓋活動頁建議之處）
- 圓角：__px（同頁一致）
- Section padding：__px
- 動態：<有哪些、時長>
- 字級微調：<若主視覺 h1 放大到 __>

## 主視覺 / 素材
- Hero：<描述 / 檔案>
- 裝飾元素：<描述>

## 破例註記（若有）
- <哪條建議 / 何種特例，為什麼，經誰同意>
```

→ 專屬規範**不可**覆蓋核心護欄；任何破例只能是「活動頁建議（§二）」層級，且要在「破例註記」寫原因。
