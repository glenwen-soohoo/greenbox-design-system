方案 C — 文青風格視覺預覽

無毒農品牌色 + 2 個延伸色（深森林綠、極淺綠）。文青風格透過字型（Noto Serif TC 標題）、版型（單欄堆疊）、間距（128px Section padding）、長寬比例（Banner 21:9 / 商品卡 4:5）、攝影風格表現。

C 風格特色速覽

項目C 文青風格的決定
主色#4ba83b 主綠（無毒農品牌色）
子品牌主色沿用各子品牌品牌色（不改）
標題字型Noto Serif TC（襯線、文青感）
內文字型Noto Sans TC
字級基準h1 48px / body 17~18px（大一階）
字重標題 500（中等，不要 600/700）
圓角0px（極簡）
Section padding（桌機）112~128px（大留白）
容器 max-width880px（精緻收窄）
主視覺 layout大量單欄堆疊
Hero 高度84~90vh
Banner 比例（桌機）21:9 寬螢幕
Banner 比例（mobile）1:1 方形
商品卡比例4:5 直立
主 CTA 樣式主綠底白字 / 0 圓角 / hover 變深綠
次要按鈕文字底線型（無框）
攝影風格極簡背景、自然光、低飽和
懸浮 FAB不用（極簡）

四、字級階梯（C 用 Noto Serif TC + 大一階）

標題用 Noto Serif TC 襯線體；內文用 Noto Sans TC；字級比 A 大一階；字重 500（中等，不要 600/700）。

五、按鈕（C 用主綠 + 0 圓角）

主 CTA 維持 A 主綠 #4ba83b，hover 變標題深綠 #008000；C 的差異在 0 圓角 + 較大 padding。次要按鈕用文字底線型。

hover 主 CTA：背景變 #008000 標題深綠（A 既有色）；hover 文字按鈕：底線變細色變主綠 #4ba83b。
