無毒農 Design System
三方案決策樣本

為了對齊 AI 時代的設計需求，我們把無毒農既有風格整理成 方案 A 踏實版，並提出 方案 C 文青版 作為視覺升級的可能方向。每個方案都附上規範總覽 + 兩個「關於無毒農」頁面範例，方便老闆直接看出選擇後的長相。

📐 2 個方案（A 踏實 / C 文青） 📄 6 個展示頁面 🌿 主色 #4ba83b
方案 A · Plan A

踏實版 — 規範化現有風格

把現有官網「不動的頁面」（商品、訂單、子品牌、Blog）的視覺規則整理成一份規則書。主色不變、字型不變、圓角 4px、Section padding 64px。最低風險、最快上線、AI 改版時可以從這份規則開始疊加。

方案 C · Plan C

文青版 — 主流 AI 預設框架

主色完全沿用無毒農品牌色（不另創），但字型升級為 Noto Serif TC、圓角全部 0px、Section padding 112~128px。對齊主流 AI 在「無提示」時自然會產生的設計風格——襯線標題 + 大留白 + 大圖。

方案對照

A 跟 C 差在哪？

兩個方案都以無毒農主色 #4ba83b 為基礎，差異全在字型、版型、間距、留白尺度上。

項目 方案 A · 踏實 方案 C · 文青
主色 #4ba83b（不變） #4ba83b（不變）+ 兩個延伸色
字型 Noto Sans TC Noto Serif TC（標題）+ Noto Sans TC（內文）
字重 700 / 600 / 500 / 400 500（中等，不要 600/700 太陽剛）
圓角 4px 0px（極簡）
Section padding（桌機） 64px 112~128px（大留白）
容器寬度 1140px 880px（精緻收窄）
Hero 高度 50/50 圖文比例 84~90vh，4:5 直立圖
Banner 比例（桌機） 16:9 21:9 寬螢幕
推估改版工程量 低（沿用既有元件） 高（攝影 brief、行銷文案、客服訓練皆需重做）
